How a CNN sees a picture

A convolutional neural network turns raw pixels into a confident answer — "this is a cat" — through a few simple, repeatable steps. Let's watch each one.

↓ Use the tabs below. Press Play on each step to watch the animation.
The Big Idea

A CNN is a team of pattern hunters.

Think of an airport security line. No single officer checks everything. One looks for edges of bags, another for shapes, another for whole objects — each passes what they find to the next. A CNN works the same way: simple detectors stack up to recognise complex things.

A CNN takes an image and passes it through a pipeline. Each stage keeps the useful information and throws away the rest, until what's left is a clear decision.

Each box is one stage. Click through the tabs to open up what happens inside.
Remember this A CNN doesn't "look" at a picture all at once. It scans for small patterns, combines them into bigger ones, and only at the very end makes a decision.
Step 1

To a computer, a picture is just numbers.

Imagine a paint-by-numbers grid. Every tiny square (a pixel) has a brightness value — 0 is black, 255 is white. A photo is just a big grid of these numbers. The computer never sees a "cat"; it sees a spreadsheet of brightness.

Hover or play to reveal the numbers hiding behind a simple image of the letter "X".

Remember this An image is a grid of numbers. Everything the CNN does next is just clever arithmetic on this grid.
Step 2

A filter slides across, hunting for one pattern.

Picture a stencil moving across a page. A small window — the filter (or kernel) — slides over the image, one patch at a time. At each spot it asks: "does this patch match the pattern I'm looking for?" A high score means "yes, found it here".

Watch the orange window slide. Each position produces one number in a new, smaller grid — the feature map — which lights up where the pattern was found.

filter window (scanning) feature map (pattern strength)
Remember this One filter detects one pattern (an edge, a curve). A real CNN runs many filters at once, building a stack of feature maps — one per pattern.
Step 3

Pooling keeps the loudest signal, drops the rest.

Like summarising a paragraph in one word. Pooling looks at a small block of the feature map and keeps only the strongest value (max pooling). The picture gets smaller and the important signals stand out — and a feature is still recognised even if it shifts a little.

Each 2×2 block on the left collapses to its single biggest number on the right.

block being summarised winning value kept
Remember this Pooling makes the network smaller, faster, and more tolerant of small shifts — the cat is still a cat if it moves two pixels left.
Step 4

The grid is unrolled into one long line.

Like unspooling a roll of film into a single strip. After convolution and pooling, we have a small grid of strong features. To feed it into the final decision-maker, we lay every value end-to-end in one row of numbers.

Watch the grid cells march out into a single line of inputs.

Remember this Flattening just reshapes the data — no information is added or removed. It turns a 2-D grid into a 1-D list ready for the final step.
Step 5

The network weighs the evidence and decides.

Like a panel of judges scoring a contestant. Every feature votes, with a learned weight, toward each possible answer. The scores are turned into percentages that add up to 100%. The highest one wins.

Watch the features connect to the output options, then settle into a confident prediction.

Remember this The "learning" in deep learning lives in these weights. Training nudges them, over millions of examples, until the right answer reliably gets the highest score.
Recap

Five steps, start to finish.

Press play to run the whole journey once more — image to answer.

In plain words: A photo is numbers → filters scan for patterns → pooling shrinks and sharpens → flatten lines it all up → the dense layer weighs every clue and picks the most likely answer. Stack these blocks deeper and the CNN learns to recognise faces, tumours, road signs — anything you give it enough examples of.
One sentence to keep A CNN recognises images by detecting simple patterns first, combining them into complex ones, and only deciding at the very end.
